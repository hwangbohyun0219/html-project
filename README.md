#### 프로젝트 소개
+ HTML, CSS, JavaScript, Vue3, Node.js와 oracle을 활용하여 로그인,아이디 중복체크와 회원가입을 하고 품목에 대한 설명을 게시하는 사이트를 만들었습니다.
+ 프로젝트를 하면서 사이트의 틀을 지키지만 나만의 느낌을 줘보자는 생각을 많이 했습니다.
